상생 (相生)

木 → 火 → 土 → 金 → 水 → 木

木生火(나무는 불을 낳고) • 火生土(불은 흙을 낳고)
土生金(흙은 쇠를 낳고) • 金生水(쇠는 물을 낳고)
水生木(물은 나무를 낳는다)

상극 (相剋)

木 → 土 → 水 → 火 → 金 → 木
